hexo 配置

  • hexo 配置文件

  • 安装git nodejs

  • 安装 hexo-cli

    npm intall hexo-cli -g

  • 安装hexo

    npm install hexo --save

  • 安装hexo-deployer-git

    npm install hexo-deployer-git --save

    安装配置

    1. type:git
    2. repo:https://github.com/luoye7121/luoye7121.github.io/
    3. branch: main
    • 要用gitbash
      hexo d时会输入用户名字和密码，但是cmd会在输入密码的时候卡住，gitbash就不会。而且如果输入密码的次数多，好像会锁定，一段时间内进不去。所以，一定要用gitbash！！！！

  • 创建ssh秘钥

    本地创建秘钥之后复制下来，在GitHub中覆盖之前存在的ssh秘钥
