  • 第一步: 测试:添加js脚本引用
    <script src="https://piccache.cnki.net/yxpt/sta_api/v1/scripts/sensors.js"></script>
  • 第一步: 开发:添加js脚本引用

业务模块初始化添加
静态类属性 a,button,input,textarea 标签仅需添加sensors-data即可

静态类属性 在标签上sensors-data 添加相应属性 属性必须是用双引号添加属性(原因json转换属性只支持双引号定义的属性)

属性 value 是否必填
app_id 项目appid true
event_id 埋点事件ID true
event_target 埋点事件名称 false
event_secondary_id 埋点次级事件ID false
event_secondary_target 埋点次级事件名称 false
